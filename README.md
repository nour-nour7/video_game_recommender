# video_game_recommender
Recommendation system for video games

App link: https://videogamerecommender.streamlit.app/


Start typing video game name, select the desired video game, click on the get recommendations button. Click on the image to view the store page, expand the description box to get more information.

Links for the datasets: 

- games dataset: https://www.kaggle.com/datasets/fronkongames/steam-games-dataset/data?select=games.json
- reviews dataset: https://www.kaggle.com/datasets/andrewmvd/steam-reviews
